---

## Instruções de Execução

Execute o makefile para gerar o arquivo executável final `corrida`:

```
make
```

Para executar o programa, execute a seguinte linha de comando no terminal:

```
./corrida 1 (utilizando barreira)
./corrida 0 (sem uso de barreira)
```
